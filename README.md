# 👟 Step Up Shop - E-commerce Mobile App

A comprehensive e-commerce mobile application for a shoe store, built using Flutter with full cloud integration for a seamless shopping experience.

## 🚀 Key Features
* User Authentication: Secure sign-in and sign-up powered by Firebase Authentication.
* Dynamic Product Catalog: Real-time display of footwear with detailed descriptions, pricing, and sizes.
* Content Management: Backend management via FireCMS for easy product updates.
* Cloud Storage: Integrated with Cloudinary for high-performance image hosting and delivery.
* Shopping Cart: Fully functional cart system allowing users to add, remove, and manage items before checkout.

## 🛠 Tech Stack
* Frontend: Flutter & Dart
* Backend: Firebase (Firestore, Auth)
* Media Management: Cloudinary
* State Management: Provider
